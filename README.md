# C ?
## What is C ?

C is a powerful language but low-level invented in 1972, it was made to build OS, memory management and more.

It is for pure speed !

## Why learn it ?

You may think if python (high-level language) exists, why use C ?

Well because they have different jobs !

Python is built to make apps but C is made to interect with the cpu with no other so called 'middle man', that is also the reason it is so fast !

Think of python as an automatic car and C as a manual one

### Fun Fact - Python was also built using C !

## Where to learn it ?

There are many free sources like freeCodeCamp or W3 Schools to get started, additionally tinker with the language, visit GitHub repos and build projects, believe it or not they have different uses but starter projects remain the same as python starter projects like a basic calculator !
